HRdepartment - Программа «Отдел кадров»

Описание
Программа для автоматизации учёта сотрудников завода «КузбассМаш». Реализованы функции: управление сотрудниками, отпусками, нарушениями, поиск, аналитика, экспорт в Excel/Word, история изменений, прикрепление файлов.

Технологии
- C# .NET Framework 4.7.2
- Windows Forms
- Microsoft SQL Server

Установка и запуск
1. Установите Microsoft SQL Server Express
2. Выполните скрипт `Database.sql` для создания базы данных
3. Откройте решение `HRdepartment.sln` в Visual Studio 2022
4. Настройте строку подключения в `App.config`
5. Запустите проект (F5)

Тестирование
- Модульные тесты находятся в проекте `UnitTest`
- Для запуска тестов: Тест → Запустить → Все тесты
